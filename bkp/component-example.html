Reusable Components Example

This page demonstrates how to use the testimonials and blog section components

How to Use Components

Step 1: Include the Component Loader Script

Add this script before the closing </body> tag:

<script src="components/component-loader.js"></script>

Step 2: Add Component Placeholders

Use data-component attribute to load components:

<div data-component="testimonials"></div> <div data-component="blog-section"></div> <div data-component="footer"></div>

Available Components

  • testimonials - Customer testimonials section with slider
  • blog-section - Latest blog posts section
  • footer - Complete footer with navigation, social links, and newsletter
